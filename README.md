# E-commerce-Olist-Store-Data-Analysis
This project presents an in-depth analysis of the Olist e-commerce dataset, covering sales trends, customer behavior, payment patterns, and delivery performance. It includes ETL pipelines, exploratory data analysis, and interactive dashboards. The goal is to uncover key business insights and support data-driven decision-making.

##  Olist-Store Analysis Dashboard
# Project Overview
This project delivers a comprehensive analysis of the Olist e-commerce dataset, exploring sales performance, customer behavior, payment trends, and delivery efficiency. Through data cleaning, transformation, and visualization, it uncovers key insights that influence business decision-making. An interactive dashboard further enhances the understanding of marketplace operations.

## Dataset Used
This project uses the Olist E-commerce Public Dataset, containing 100K+ orders with customer demographics, product categories, payment methods, delivery performance, and customer review data.
